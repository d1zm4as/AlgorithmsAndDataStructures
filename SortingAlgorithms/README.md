# SortingAlgorithms

Collection of sorting algorithm implementations.

## Structure

- `ElementarySortingAlgorithms`
  - `bubblesort.c`
  - `insertionSort.c`
  - `selectionSort.c`
- `EfficientSortingAlgorithms`
  - `quick_sort.c`
  - `mergeSort.c`
  - `heapSort.c`
  - `counting_sort.c`
  - `bucket_sort.c`
  - `radix_sort.c`
  - `shell_sort.c`

## Build example

```bash
gcc EfficientSortingAlgorithms/quick_sort.c -o quick_sort
./quick_sort
```

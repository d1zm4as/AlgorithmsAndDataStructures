# AlgorithmsAndDataStructures

Study repository for core data structures, graph theory, sorting algorithms, and core algorithm techniques.

## Structure

- `DataStructures`: linear and foundational structures.
- `GraphTheory`: traversal and shortest path style algorithms.
- `SortingAlgorithms`: elementary and efficient sorting implementations.
- `SearchingAlgorithms`: classic search techniques.
- `DynamicProgramming`: DP fundamentals and problems.
- `GreedyAlgorithms`: greedy strategy algorithms.
- `StringAlgorithms`: string matching algorithms.

## Languages

Mainly C and Python.
